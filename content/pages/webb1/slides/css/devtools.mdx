# Developer tools

---

Övning:

1. Gå in på valfri nyhetssida (tex Aftonbladet, HelaGotland, Expressen)
2. Öppna upp developer tools och börja ändra innehållet och stilen på hemsidan
   Till exempel ändra rubriker och textinnehållet så det står något annat, ändra textfärgen, bakgrunden, osv
3. Ta en skärmdump (Windows-tangenten+Prt Scr [Fn+Insert]) och spara bilden till senare.
