Unity är en spelmotor.

## Hämta Unity

1. Gå till [Unitys hemsida](https://unity.com/).
2. Klicka på Get Started-knappen, och sedan tabben Individual.
3. Under rubriken Personal, klicka på Get Started-knappen.
4. Under rubriken First-time users, klicka på Start here-knappen, och sedan Agree and download-knappen.
5. Under tiden du laddar ner, klicka på profilbilden längst upp till höger på samma sida, och välj Create a Unity ID. Följ instruktionerna för att skapa ett Unity ID, du behöver det för att använda Unity.
6. Installera Unity.
7. Starta Unity Hub, logga in och hämta en Personal license.

## Använd VS Code som kodeditor till Unity

Gå till [Unity Development with VS Code](https://code.visualstudio.com/docs/other/unity) och följ instruktionerna för Windows.
