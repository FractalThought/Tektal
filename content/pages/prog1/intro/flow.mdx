# Öva på flödet

Vi startar ett projekt genom att öppna upp Visual Studio Code och navigera till vår projektfolder, och skapar en ny folder (intro) som vi öppnar.

I denna foldern skapar vi en ny fil som vi döper till main.js, och öppnar den filen.

Här kan vi nu skriva kod, och köra den. Vi börjar med att testa Hello World:
<a id="hello-world"></a>

## Hello World

Vi skriver `console.log("Hello World");` i filen och sparar med kortkommandot Ctrl+S.

Sedan visar vi upp terminalen/konsolen (View > Terminal).

Slutligen går vi till terminalen och skriver in `node main.js` (filens namn)
<a id="flera-rader"></a>

## Flera rader

Vi kan kopiera denna raden kod genom att markera hela raden (med musen, eller med kortkommandot Ctrl+Shift+Pil höger/vänster några gånger [Kortkommandot markerar hela ord istället för karaktär]). Använd kortkommandot Ctrl+C för att kopiera det du markerat.

Skapa sedan en ny rad (Entertangenten) och klistra in din kopierade rad med kortkommandot Ctrl+V.

Vi kan också duplicera raden med kortkommandot Alt+Shift+Pil upp eller Pil ner.

Ändra nu så att andra raden säger något annat, till exempel ditt namn.

Det gör du genom att ta bort Hello World inuti citattecknen ("") och skriva något annat där istället.

Gå sedan tillbaka till terminalen och skriv in `node main.js` (filens namn).

Notera vad som händer - ett flöde sker där datorn kör en rad efter den andra.

Upprepa och testa på att använda kortkommandot eller kopiera och klistra in några gånger. Bli bekväm med detta då du kommer använda det väldigt mycket.
<a id="byta-ordning"></a>

## Byt ordning på raderna

Nu när vi har fler rader så kan vi testa vad som händer när vi byter ordningen på raderna.

Du kan göra på liknande sätt som du gjorde tidigare - markera hela raden och använd kortkommandot Ctrl+X istället för C för att klistra ut. Gå sedan upp med piltangenterna och klistra in med kortkommandot Ctrl+V igen.

Om du vant dig med kortkommandot Alt+Shift+pil upp/ner för att duplicera en rad, så kan du testa att skippa Shift (alltså bara Alt+pil upp/ner) för att flytta en rad. Det är ett väldigt effektivt och snabbt sätt att arbeta på.
