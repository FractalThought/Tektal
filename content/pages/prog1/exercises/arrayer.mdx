# 1: Kö till Lustiga Huset
Kalle och hans syskon vill gå in i Lustiga Huset, men tivolit har en begränsning av hur många som får vara inne i huset, så det blir en kö utanför.

I kön ser Kalle hans vänner. De står i följande ordning:
Igge, Isabella, Isak, Kalle, Kasper, Klara, Saga, Sture, Sara.

Skapa en array fylld med namnen på alla som står i kön, och skriv sedan ut vad varje person har för plats i kön.

<PracticeContainer>

```javascript
let peopleInLine = ["Igge", "Isabella", "Isak", "Kalle", "Kasper", "Klara", "Saga", "Sture", "Sara"];

for(let i = 0; i < peopleInLine.length; i++) {
        console.log(peopleInLine + " är på plats" + i + " i kön");
}

```

</PracticeContainer>

# 2: Lista på lottorader
Tivolit behöver skapa en lista på lottorader och spara dem. Tips: Kolla iterationssatser.

<PracticeContainer>

```javascript
let = numberOfTickets = 5; // Ändra för att få fler lottorader
let numberInTickets = 5; // ändra för att få fler siffror i varje rad
let tickets = []; // våra sparade lottorader

// Loopa igenom för att skapa flera tickets
for (let i = 0; i < numberOfTickets; i++) {

        // Skapa en ticket genom att loopa igenom 5 gånger och lägga till en slumpmässig siffra
        let ticket = "";
        for (let j = 0; j < numberInTickets; j++) {
                ticket += Math.floor(Math.random() * 10) + " ";
        }
        // Lagra lottoraden vi skapat
        tickets[i] = ticket;
        console.log("Ticket " + i + ": " + tickets[i]);
}
```

</PracticeContainer>
